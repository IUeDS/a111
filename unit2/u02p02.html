Any conductor has some amount of resistance. There are also components called resistors that have a specific resistance. Resistors are the first and most common type of electronic component. They make up the majority of most electronic circuits. Figure 2-1 shows a number of different actual resistors. The schematic symbol used for resistors is shown in Figure 2-2.

Most resistors are labeled with colored bands to indicate the resistance value. This is because historically they were physically too small to allow a value to be written on them. Today the resistors have gotten even smaller but it is now possible to print a numeric resistance value on them and you will see that with some brands of resistors. However, it is still most common to see the colored bands to indicate the resistance value.

Historically there have been four bands, though it is now common to see five or even six bands. Each band gives some information about the resistance value of the resistor. Four and five band resistor color codes are separated as shown in the tables below. A six band color code would simply add fourth digit indicator band.

4 band color code
1st digit 2nd digit Multiplier Tolerance
Value = 1st digit, 2nd digit, x multiplier
5 band color code
1st digit 2nd digit 3rd digit Multiplier Tolerance
Value = 1st digit, 2nd digit,3rd digit x multiplier

The first two to three bands are the digit indicators and follow the color code in the table below. These digit indicators give you the numbers that come before the multiplier to give you the resistance value. An example is shown in Figure 2-3.

The last band is for tolerance, there is additional space between it and the other bands to help differentiate it.
It is difficult to make a resistor at an exact value, so they come in different tolerances. Tolerance indicates how close + or – the actual value of the resistor is to its stated value. For instance, a 1KΩ

10% resistor could be anywhere from 900Ω to 1100Ω. The tighter the tolerance, the more the resistor will cost. Tolerance is indicated by the color code in the table below. The example is once again shown in Figure 2-5.
