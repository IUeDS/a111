If we want to build a circuit that creates a high pass filter we can now use either an inductor or a capacitor. In either of these cases it is a first order filter. As we learned in Chapter 3, the slope of the filter is set at $ \frac {6dB}{8_{VA}}$. That means six decibels per octave.

  • Decibel ($dB$): A logarithmic way to define values. This is used a great deal with audio as it emulates the way that human beings hear. The math is actually not too complex. To find a $dB$ value you need to compare two things. Often that is a set reference value but in this case we are just comparing two values. To find the slope of a filter you would compare the voltage at a frequency with the voltage at a different frequency. We will be using two frequencies that are an octave (see below) apart. The formula when you are starting with voltage is $dB = 20log(\frac {V1}{V2})$

  • Octave($8_{VA}$): An octave is a musical interval that equates to a doubling of frequency. A frequency that is an octave above would be twice the frequency. Likewise, an octave below would be half the frequency.


If we want a slope that is more than $ \frac {6dB}{8_{VA}}$ we get it by adding another filter that has the same affect.

For instance, a capacitor in series with the load and an inductor in parallel with the load would create a second order high pass filter. This circuit, as shown in Figure 9-1. would have a slope of $ \frac {12dB}{8_{VA}}$ . The frequency response curve for this is also shown in Figure 9-1.

If you reversed the inductor and capacitor (L in series, C in parallel) you would have a $ \frac {12dB}{8_{VA}}$ low pass filter.

Changing Values

Our old rule for changing inductance and capacitance still applies here. As we increase capacitance and inductance the curve will move to the left, as we decrease capacitance and inductance the curve will move to the right.
