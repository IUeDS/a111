Because reactance changes with frequency it is often necessary to analyze a circuit for multiple frequencies. You can do this by creating equivalent circuits at multiple frequencies. Once you have equivalent circuits, you can use Ohms law to find voltage drop or current flow.

For simple series or parallel inductor circuits you can typically find reactance for three frequencies (such as: $ 100Hz, 1kHz,$ and $10kHz $, for easy math). Use the reactance value you get to replace each inductor in the circuit with a resistor for that specific frequency. You then end up with three voltage divider circuits (one for each frequency) to analyze. From those you find the voltage across the load for each of them and you can plot the frequency response on a graph.

Let’s try that with the circuit in figure 8-1.
First we solve for inductive reactance at $10kHz, 1kHz,$ and $100Hz$.

$10kHz$:

$X_L = 2 \times \pi \times 10,000 \times .159 = 9990.2\Omega $
or
$X_L = 2 \times 3.14 \times 10,000 \times .159 = 9985.2\Omega $

For this sort of analysis we can round both of these up to $10K\Omega $.


$1kHz$:

$X_L = 2 \times \pi \times 1000 \times .159 = 999\Omega $
or
$X_L = 2 \times 3.14 \times 1000 \times .159 = 998.5\Omega $

Again we can round both of these up to $1KΩ$.


$100Hz$:

$X_L = 2 \times \pi \times 100 \times .159 = 99.9\Omega $
or
$X_L = 2 \times 3.14 \times 100 \times .159 = 99.85\Omega $

Again we can round both of these up to $100\Omega $.


A common mistake is to forget that the formula for reactance uses: henries, and hertz. In this example we have:
  • $159mH$ so we have to convert to $0.159H$
  • $10kHz$ and $1kHz$ which we have to covert to $10000Hz$ and $1000Hz$.
However, we can use $ mH $ as long as we use $ k\Omega $

Next you can solve for the voltage drop of each circuit. RL is our load for this circuit so we are concerned with the voltage that it drops.

Using Amperes

For $10kHz$ the inductor becomes a $10k\Omega $ resistor.

$11 V_{RMS} = 11,000\Omega \times ?A $

so

$A = .001A$

$R_L$ is $1k\Omega $

so

$V_{RL} = 1,000\Omega \times .001A $

so

$V_{RL} = 1V_{RMS}$

Using Milliamperes

For $10kHz$ the inductor becomes a $10k\Omega $ resistor.

$11 V_{RMS} = 11k\Omega \times ?mA $

so

$A = 1mA$

$R_L$ is $1k\Omega $

so

$V_{RL} = 1k\Omega \times 1mA$

so

$V_{RL} = 1V_{RMS}$

For $10kHz$ the inductor becomes a $10k\Omega $ resistor.

$11 V_{RMS} = 11,000\Omega \times ?A $

so

$A = .001A$

or

$11 V_{RMS} = 11k\Omega \times ?mA $

so

$A = 1mA$

$R_L$ is $1k\Omega $

so

$V_{RL} = 1,000\Omega \times .001A $

so

$V_{RL} = 1V_{RMS}$

or (preferred)

$R_L$ is $1k\Omega$

so

$V_{RL} = 1k\Omega \times 1mA$

so

$V_{RL} = 1V_{RMS}$
