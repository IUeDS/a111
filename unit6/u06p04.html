Starting with the power supply makes sense as it is required for all other sections of the circuit. After that a good method is to follow the audio. The audio starts at the input and finished at the output so we will go in that same order.

Figure 4-1 shows a re-drawn close up of the input section. This amplifier has two input connections or "jacks". You can either use them at the same time with two different instruments or choose which to use for a single instrument. In the case of this amplifier, the two inputs will create two different outputs if used separately.

You may notice that the schematic symbols next to 1 and 2 on the schematic are not something we have seen before. These are symbols for ¼” jacks. These are used for electric guitars. Figure 4-2 shows these schematic symbols.

 

There are two connections shown for this schematic symbol. The "positive" connection is where our audio comes in when a plug is inserted into the jack. The "switched" connection changes depending on whether something is plugged in or not. When something is not plugged in to this jack the "switched" connection is connected to the "positive" connection. When something is plugged in, this connection is broken. More on how this is used coming up next. There is also a ground connection for this jack that is not shown; it must be connected to ground in the circuit to function properly.

Plugged into #1 only

If something is plugged into input #1 but nothing is plugged into input #2 the circuit can be re-drawn as shown in Figure 4-3.

Since nothing is plugged in to input #2, the switched connection connects the two $68k\Omega$ resistors in parallel. This could be simplified into a single $34k\Omega$ resistor and is in series with the audio signal. There is also a $1M\Omega$ resistor from input #1 to ground. Since there it very little current into the control grid of a tube, the $34k\Omega$ resistor will drop very little voltage. Almost all of the voltage from the instrument will get to the control grid.

Let’s see how different it would be with something plugged into #2 instead.

Plugged into #2 only

Figure 4-4 shows the circuit re-drawn if something was plugged into only input #2. With nothing plugged in to input #2, the $1M\Omega$ resistor is shorted out and we have another voltage divider. This time the two resistors are both $68k\Omega$. that means that half of the input voltage will get to the control grid. We now know that plugging into input #1 by itself will create a louder sound than plugging into input #2 by itself.

Playing with a Friend?

What about plugging into both inputs with different instruments? Figure 4-5 shows this option. In this case the signal from both inputs must path through a $68k\Omega$ resistor before getting to the input. In the case of input #1 there is also a $1M\Omega$ resistor to ground. We know that there will be next to no current flowing into the control grid of the triode; that means that the $68k\Omega$ resistors will drop next to no voltage. The entire voltage from both input #1 and input #2 will get to the control grid.

A note about this process

This may have seemed like a long drawn out process to figure out what we did. The idea of re-drawing the circuit for different situations is a very important one to learn and practice, so it is worth taking the time. We will explore this more in Chapter 15.
