If we simply want to find a numeric value for an AC waveform we can use a multi-meter. That wonâ€™t give us any information about what kind of waveform it is however. For that we need to actually see the waveform as it changes over time. For this we use an oscilloscope.

An oscilloscope is a voltmeter with two connections per channel. The oscilloscope must be connected in parallel, just like any other voltmeter. There may be more than one channel which allows you to compare two waveforms.

Oscilloscopes create an image on an X/Y axis where X (left to right) represents change over time and Y (top to bottom) represents voltage. In Figure 16-1 we see a sine wave displayed on an oscilloscope. We can set the divisions of the X and Y axis to allow the image to be "zoomed" in or out.
