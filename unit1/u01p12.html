We know that voltage drops in a series circuit add up to the voltage source, while in a parallel circuit, the voltage is the same for all branches of the circuit. For current it is the inverse.

Current in a series circuit

Since every electron that goes through one component in a series circuit must go through the others, the current through each component will be the same. That is the case; current is the same through all components connected in series. In the series light bulb circuit, shown in Figure 12-1, both light bulbs get the same amount of current and each drops half of the source voltage.

Current in a parallel circuit:

Electrons can choose to go through either component in a parallel circuit so the current through both add up to be the total current that comes from the source. If, as in our light bulb circuit, the branches are exactly the same, then the total current will be twice the current through each light bulb. This is shown in figure 12-2. The voltage dropped by each light bulb will be equal to the source voltage.
